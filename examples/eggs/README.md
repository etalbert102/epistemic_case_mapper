# Eggs And Health Example

Evidence mode: `source_grounded`

Review status: `draft`

Source manifest: `data/cases/eggs/case.yaml`

Generation command:

```bash
PYTHONPATH=src python3 scripts/build_case_map.py --case data/cases/eggs/case.yaml --output-root examples
```

Validation command:

```bash
PYTHONPATH=src python3 scripts/build_case_map.py --case data/cases/eggs/case.yaml
PYTHONPATH=src python3 scripts/validate_case_artifact.py --case data/cases/eggs/case.yaml --examples examples/eggs
```

## Status

This is a source-grounded draft example. It demonstrates the artifact shape, audit surface, source-local corpus, and regeneration workflow for a messy nutrition-evidence case.

## Known Limitations

- Claims are heuristic candidates and require audit.
- Relations are generated by starter heuristics and should not be treated as settled.
- The text corpus is local, but source-span precision is still coarse.
- Direct Dietary Guidelines PDF retrieval stalled during acquisition; guideline context is currently represented by AHA and Nordic Nutrition Recommendations sources.
- Human review has not occurred.
